Títulos, paragrafos e algumas dicas:

A tag <h1> é um título de grau 1, existem 6 niveis de títulos. Niveis de títulos tem sua importância, não se usa como forma de tamanho de texto, pois isso é feito no 'CSS' onde vamos fazer o design do site. Então você não vai usar a tag <h3> porque acha que o tamanho é melhor. São niveis para serem usados como títulos e subtítulos e na hora de trabalhar com seu arquivo, ser bem mais pratico e profissional.

A tag <p> cria um paragrafo.(São os paragrafos que você está lendo) Você pode escrever o paragrafo de qualquer jeito, basta colocar no meio do par de tags <p> </p>.

Se precisar querbrar o texto em algum lugar específico, você pode usar a tag <br>.

Para colocar o sinal de maior (<) ou menor (>) no HTML, basta colocar assim '& l t ;' (Tudo junto) para o maior e'& g t ;' (Tudo junto) para menor.

A tag <hr> cria uma linha horizontal.(Igual a linha acima)

A tag <br> quebra a linha do texto.(Pois se der enter não vai fazer nenhuma diferença no texto)

Envelopar:

Uma dica para deixar um texto entre tags é usando um atalho. Selecione o texto no seu 'Visual Studio Code',em seguida no seu teclado aperte 'ctrl' 'shift' 'p', aparecerá uma caixa de pesquisa e lá você busca 'abb'. Vão aparecer algumas opções e você irá escolher a opção 'Wrap with Abbreviation' em seguida vai buscar 'p' e dar enter. Agora toda vez que precisar colocar algum texto entre tags é só selecionar o texto, usar esse atalho:'ctrl' 'shift' 'p', dar 'enter'e escrever a tag desejada e dar 'enter' novamente.

Temos a tag <code> que deixa o texto com uma formatação mais fácil de se ler. São usadas geralmente para mostrar códigos de progamação

No arquivo HTML está todo escrito com a formatação correta, mas quando abre o arquivo no navegador, ele não fica com a formatação apenas usando a tag <code>, então usamos a tag <pre> para fazer o navegador abrir da forma que escrevemos no arquivo. Ex acima, mas agora entre as tags <pre>: 

num = int(input('Digite um número:'))
    if num % 2 == 0:
        print(f'O número {num} é PAR')

        pront(f'O número {num} é IMPAR')

isso acontende porque a tag <pre> usa a pre-formatação do seu arquivo HTML.

Tamanho de imagens:

Existem vários tipos de imagens, por exemplo o .jpg o diferencial desse formato é que tem uma auta compactação. O .png tem uma compactação um pouco menor mas permite transparência. Também tem o farmato .gif com baixa compactação, mas permite animacão e transparência.

Tamanho de imagens é um assunto muito importante, o grande professór Guanabara falou: Uma imagem de largura de 1500 megas é mais que o suficiente para usar em sites.

Obs: Quando editar o tamanho de imagens você precisa redimencionar proporcionalmente o tamanho e largura. Normalmente imagens usadas em sites tem 600 megas de tamanho, que é um tamanho ótimo.

Então é bom sempre deixar seus arquivos de imagens sempre com tamanhos pequenos, para não ser um site tão pesado.

Imagens que se adaptam sózinhas:

Compatível com todos os dispositivos:

Hoje se usam muitos dispositivos para navegar na internet. Celuláres, teblets e computadores são exemplos. A diferença deles são o tamanho de cada um, e para ter uma melhor foma de navegar em um site é importante ter imagens que se adaptem a cada tamanho. Como exemplo fiz três imagens de tamanhos diferentes, uma de 300 x 300, outra de 700 x 700 e a última 850 x 850. Vamos usar a tag <picture> e dentro vamos usar as tags de imagem <img>. Usamos a tag picture para informar que vai ter imagem nesse lugar do arquivo. E essa tag Também permite usar várias fontes de imagens, precisamos dessa função pois precisaremos que a imagem mude de acordo com o tamanho. Nesse exemplo, como falado, vamos usar três tamanhos. A imagem abaixo é adaptável. Pode aumentar e diminuir seu navegador para testar.

Foi feito exatamente assim: 

<picture>
    <source media="(max-width: 700px)" srcset="imagens/P.png" type="image/png">
    <source media="(max-width: 1050px)" srcset="imagens/M.png" type="image/png">
    <img src="imagens/G.png" alt="imagem flexivel">
</picture>

Olhando para o código acima podemos ver que foi usado a tag <picture> e em seguida a tag <source>. Mas primeiro coloquei a tag <img> com a maior imagem criada 1000 x 1000. Em seguida o visual studio code quando digitamos a iniciais da palavra source vão aparecer algumas opções, você irá escolher a source media type, quando selecionar essa opção, vai auto preencher da seguinte forma:

<source media="(min-width: )" srcset="" type="image/">

Podemos observar que auto preenche com o min-width. Foi auterado para o máximo que é a maneira que acho mais fácil de usar: max-width, logo em seguida preencha o tamamho máximo permitido para que não tenha barra de rolagem na sua imagem (Nesse caso foi de 1050px). No próximo parâmetro você colocará sua imagem média. Infelizmente não consegui buscar com o ctrl espaço, mas já foi mostrado como faz para colocar imagens. E por ultimo, no parâmetro type você vai colocar o tipo de imagem que foi usado.(No caso foi usado o png)

Observe que o source para troca de imagem de grande para média foi em cima da tag <img>. O source da imagem pequena será em cima do source anterior, como demonstrado no exemplo. Será da mesma forma, mas com o whith máximo 700px e a imagem que sera posta será a pequena que foi renomeada assim: p.png. E está na pasta imagens.

Áudio no HTML:

Usar áudio:

Como adicionar:

Podemos adicionar áudios no nosso site também, usando a tag <audio>. Aqui vai o exemplo:

<audio src="exemplodeáudio.mp3">

Assim seu áudio será inserido no seu arquivo, mas não vai tocar. Para tocar você pode colocar os parâmetros autoplay que vai iniciar o áudio assim que o site for carregado ou o controls que mostrará um controle de áudio. Dessa forma:

<audio src="audios/audio1.mp3" autoplay>

<audio src="audios/audio1.mp3" controls>

Qaundo for usado o autoplay temos outra configuração que pode ser interesante na hora de colocar uma música de fundo no seu site. o parâmetro loop faz o seu áudio rainiciar sem parar, é bem comum ser usado quando o áudio é curto. Ex:

<audio src="áudios/áudio1.mp3" autoplay loop>

Dependendo do navegador o áudio não irá funcionar, mas temos como resolver esse problema usando autros formatos de áudio. Temos os formatos OGG, WAV e o que usamos de exemplo MP3. Você converte o áudio para esses três tipos e coloca no seu arquivo. Assim, se o navegardor não for conpativel com um, será com o outro. Para colocar os três formatos temos que usar o type. Quando usamos a tag <audio> ela se auto preenche com o src.

<audio src=""></audio>

Apagamos esse src e vamos dar um enter entre as tags de abertura e fechamento para colocarmos o source. Dessa forma:

<audio>

</audio>

E vamos colocar manualmente o source. Vimos na parte de imagens no HTML que enquanto estamos digitando a palavra a no seu visual studio code, vão aparecendo várias opções e você irá escolher a opção source src. Dessa forma:

<audio>
    <source src="" type="">
</audio>

Agora preencha com os áudios que você converteu e o tipo de cada um. Dessa forma:

<audio>

    <source src="exemplo3.wav" type="audio/wav"a

Obs: Para navegar no site em busca do que você quer, depois que pesquisou o tipo do arquivo, basta ficar clicando no enter até achar.

Listas:

Tipos de listas:

Listas ordenadas:

As listas ordenadas servem para que se possa alterar qualquer posisão da ordem dela sem ter tanto trabalho. Por exemplo:

1 escovar os dentes 2 merendar 3 tomar banho 4 ligar para minha mãe

Se precisar adicionar outro item, e ele precisa estar na terceira posição, você vai ter que adicionar e mudar a ordem de todos os itens abaixo. Na lista do exemplo acima não parece tão trabalhoso, mas pensa numa lista com 100 itens e precisa colocar um na posição 3. Nesse caso seria desesperador não é?

Por isso temos a lista ordenada, que usa a tag <ol>. Vamos abrir a tag e dentro dela cada item da lista ficará com a tag <il>.

Ex: Cores que mais gosto:

  1. azul
  2. cinza
  3. laranja
  4. preto

O código fica exatamente assim:

<ol>
    <li>azul</li>
    <li>cinza</li>
    <li>laranja</li>
    <li>preto</li>
</ol>

Assim sempre que precisar mudar a precedência da lista é só colocar na posição desejada, que os números vão mudar automaticamente.

Dentro da tag <ol> você pode colocar o parâmetro type, que vão ter 5 tipos:

Tipo 1: Representado pelo número 1. É o padrão, e vai ser uma lista númerica. Tipo 2: Que é representado pela letra 'A' maiúscula, é uma lista alfabetica com letras maiúsculas. Tipo 3: É representado pela letra 'a' minúscula e vai fazer uma lista com letras minúsculas. Tipo 4: É a letra 'I' maiúscula e irá fazer uma lista com os algarismos romanos. Tipo 5: É representado com a letra i minúscula e Também irá fazer uma lista com os algarismos romanos só que de um tamanho menor.

Listas não ordenadas:

São listas que no título já deixa claro, não tem ordem. A tag usada para fazer esse tipo de lista é a <ul>.

Vai ficar exatamente assim:

<ul>

    <li>cinza</li>

</ul>

O navegador pade dar a "rel" caso não coloque, então é muito importante o usó desse parâmetro.

Usando links internos:

Igual como os links externos, basta usar a tag <a> e colocar a url do seu novo arquivo. Uma forma mais fácil de fazer isso é usando o nome do arquivo que você criou. No link da minha segunda página foi feito assim:

<a href="pag002.html" rel="next"> segunda Página.>

Também pode criar um novo arquivo em uma pasta diferente, mas não poderá simplesmente colocar o nome do arquivorecém criado, como no exemplo anterior, pois dará erro. Para resolver esse erro basta usar o atalho 'ctrl' 'espaço' que vão aparecer todos os arquivos e pastas que você criou. Busque o arquivo abrindo as pastas e selecione seu link para funcionar.

Para entender melhor, fiz a uma pasta com o nome 'maispg' e criei a 'pag003.html'. Agora veja como ficou: 

<a href="maispg/pag003.html">Mais páginas.>
Fica com o nome da pasta antes do nome do arquivo, mostrando o caminho para o sistema achar o arquivo solicitado.

Links para download:

Fazer links para dawnload de PDF é muito simples, usa-se a tag <a> Também, mas possa ser que seu navegador não baixe seu arquivo e apenas o mostre, para baixar você pode clicar no segundo botão do mouse e baixar link. Pode Também tentar usar o parâmetro download com o nome do arquivo que quer baixar, se não funcionar tem mais uma configuração, o parâmetro type que usa application/pdf para mídia em pdf.

Vou exemplificar com o PDF do cursó em video para download, que foi feito exatamene assim: 

<a href="pdf/pdfdelinks.pdf" download="pdfdelinks.pdf" type="application/pdf">

Para saber qual configuração usar em cada media type basta buscar nesse link o tipo de mídia que irá usar. CLica no link, então no teclado aperta ctrl f vai aparecer uma barra de busca e é só buscar o tipo de arquivo que está usando clicando no enter. Obs: Para navegar no site em busca do que você quer, depois que pesquisou o tipo do arquivo, basta ficar clicando no enter até achar.

Você Também pode compactar seu PDF em ZIP e usar as seguintes configurações nos parâmetros já citados: 

download="o nome do arquivo zipado" type="application/zip"
Dessa forma o seu navegador Também vai baixar o arquivo do link.

CSS básico:

Três formas de usar o css:

Estilo inline:

É o menos recomendado porque dá muito trabalho e é pouco produtivo. Nessa forma você vai mudar o estilo na própria tag. Para usar exemplos mais faceis de serem entendidos, vamos colocar em páginas diferentes para vermos como ficam as mudanças. No primeiro exemplo vamos na tag <body> e dentro dela vamos colocar um parâmetro style Nesse link

Estilos locais ou internos:

Aqui vamos usar uma forma bem mais eficaz de mudar o estilo do seu site. Na cabeça do seu arquivo html vamos usar a tag <body>. Como no exemplo anterior faremos isso em uma página nova.

Estilos externos:

Como comentado na página do exemplo de estilos locais, o CSS usa muitas linhas para deixar seu site bonito. E seu arquivo HTML pode ser bem grande, Nesse caso é melhor usar o estilo externo para ficar organizado. O estilo externo também serve quando você tem vários sites e o estilo são bem parecidos.

vamos entender como funciona, no lugar de colocarmos uma tag style na cabeça do arquivo, nos vamos criar um link de css. Você só precisa ir no mesmo lugar onde criamos a tag style escrever link e vão aperecer várias opções, escolha o link css. vai auto preencher da seguinte forma: 

<link rel="stylesheet" href="style.css">

Você pode tentar entrar, mas ainda não vai funcionar. Para funcionar você terá que criar esse link,. Precisa apenas segurar o botão crtl e clicar em cima do link. Nessa hora vai aparecer uma notificação que o link não existe, agora basta clicar em criar link.

Agora tudo que você estava fazendo na cabeça do seu arquivo poderá ser feito em outro arquivo separado.

Medidas:

Medidas absolutas e relativas:

cm (centimetros), mm (milimetros), in (polegada), px (pixel), pt (ponto), pc (paica). São as medidas absolutas.

E em (Medida relativa ao tamanho M da fonte), ex (Medida relativa ao tamanho x da fonte), rem (Medida Também ralativa ao M, mas considarando o que está no CSS), vw (Medida relativa a largura da tela do dispositivo usasdo), vh (Medida relativa a altura da tela do dispositivo usado), % (Medida relativa a porcentagem). São as medidas relativas.

dica:

É recomendado usar as medidas em e px.
