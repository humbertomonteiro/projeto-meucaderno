Estilo locais ou internos:

Mais eficaz

Como visto na página anterior precisamos abrir uma tag <style> na cabeça do seu arquivo HTML. Normalmente faço abaixo da tag <title>. Da seguinte forma:

        <head>
            <meta charset="UTF-8">
            <meta http-equiv="X-UA-Compatible" content="IE=edge">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Estilos locais</title>
            <style>
                
            </style>
        </head>

Tudo dentro dessa tag é CSS, já sabemos que No HTML usamos tags, já no css usamos seletores, que são muito simples de serem feitos. Basta que dentro da tag você escreva o seletor que são as tags do HTML, abra chave, automaticamento o visual studio code fechará a chave. (Chaves {}) e use as configurações que quiser dentro dessas chaves.

Vamos fazer as mesmas configurações que o estilo inline e veremos que tudo vai ficar mais simples de entender e bem mais organizado.

        <style>
            body {
                background-color: aquamarine;
                font-family: Arial, Helvetica, sans-serif;
                font-size: 15px;
            }
        </style>

O estilo vai tomar muitas linhas de código, e por isso essa forma não será a melhor, Vamos usar a terceira tecnica de estilo para acabar com esse problema.
