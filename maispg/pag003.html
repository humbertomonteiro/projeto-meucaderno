Uma boa dica:

Links de terceiros como visto na leitura da página anterior, se abertos sobrepondo o seu site, dificilmente voltarão para sua págian novamente, então é importante sempre colocar uma target="_blank" para ser aberto em outra aba.

Mas quando são suas próprias páginas a dica é continuar usando o target, só que em vez de usar _blank você usará o _self que significa mesma aba e abrirá na mesma aba do seu navegador, evitando que quem está navegando fique com muitas abas abertas.

Agora para voltar para primeira página que está numa pasta diferente você terá que voltar uma pasta ou dará erro.

Para resolver é muito simples, use a tag <a> com isso vamos usar "./" para mesma pasta, "../" para pasta anterior. Nesse caso usaremos o "../" pos queremos acessar a página anterior. Para exemplificar veja como fiz para retornar para página anterior que está em outra pasta: 

<a href="../index.html">Primeira página.
