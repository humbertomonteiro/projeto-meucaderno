Listas:

Tipos de listas:

Listas ordenadas:

As listas ordenadas servem para que se possa alterar qualquer posisão da ordem dela sem ter tanto trabalho. Por exemplo:

1 escovar os dentes 2 merendar 3 tomar banho 4 ligar para minha mãe

Se precisar adicionar outro item, e ele precisa estar na terceira posição, você vai ter que adicionar e mudar a ordem de todos os itens abaixo. Na lista do exemplo acima não parece tão trabalhoso, mas pensa numa lista com 100 itens e precisa colocar um na posição 3. Nesse caso seria desesperador não é?

Por isso temos a lista ordenada, que usa a tag <ol>. Vamos abrir a tag e dentro dela cada item da lista ficará com a tag <il>.

Ex: Cores que mais gosto:

  1. azul
  2. cinza
  3. laranja
  4. preto

Assim sempre que precisar mudar a precedência da lista é só colocar na posição desejada, que os números vão mudar automaticamente.

Dentro da tag <ol> você pode colocar o parâmetro type, que vão ter 5 tipos:

Tipo 1: Representado pelo número 1. É o padrão, e vai ser uma lista númerica. Tipo 2: Que é representado pela letra 'A' maiúscula, é uma lista alfabetica com letras maiúsculas. Tipo 3: É representado pela letra 'a' minúscula e vai fazer uma lista com letras minúsculas. Tipo 4: É a letra 'I' maiúscula e irá fazer uma lista com os algarismos romanos. Tipo 5: É representado com a letra i minúscula e Também irá fazer uma lista com os algarismos romanos só que de um tamanho menor.

Listas não ordenadas:

São listas que no título já deixa claro, não tem ordem. A tag usada para fazer esse tipo de lista é a <ul>.

Dentro da tag <ul> você Também pode colocar o parâmetro type, que vão ter 3 tipos:

"disc"(que é o padrão):

  • azul
  • cinza
  • laranja
  • preto

"circle":

  • azul
  • cinza
  • laranja
  • preto

"square":

  • azul
  • cinza
  • laranja
  • preto

Dica: Usar indentação

A indentação de lista é uma boa opção para organizar listas, pode usar tag <ol> junto com tag <ul>, tanto faz a ordem. isso faz suas listas mais fáceis e bem mais trabalhadas.

  1. Fortaleza
    • Vermelho
    • Azul
    • Branco
  2. Ceará
    • Preto
    • Branco

Lista de Definições:

Nesse tipo de lista abrimos com a tag <dl> com isso abrimos o termo com a tag <dt> e a definição com a tag <dd>. Ex:

HTML
Linguagem de marcação para a criação do conteúdo de um site.
CSS
Linguagem de marcação para a criação do design de um site.
jávaScript
Linguagem de programação para a criação de interatividade de um site.

Fica como um dicionário, com o termo e a definição.
