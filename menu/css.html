CSS básico:

Três formas de usar o css:

Estilo inline:

É o menos recomendado porque dá muito trabalho e é pouco produtivo. Nessa forma você vai mudar o estilo na própria tag. Para usar exemplos mais faceis de serem entendidos, vamos colocar em páginas diferentes para vermos como ficam as mudanças. No primeiro exemplo vamos na tag <body> e dentro dela vamos colocar um parâmetro style Nesse link

Estilos locais ou internos:

Aqui vamos usar uma forma bem mais eficaz de mudar o estilo do seu site. Na cabeça do seu arquivo html vamos usar a tag <body>. Como no exemplo anterior faremos isso em uma página nova.

Estilos externos:

Como comentado na página do exemplo de estilos locais, o CSS usa muitas linhas para deixar seu site bonito. E seu arquivo HTML pode ser bem grande, Nesse caso é melhor usar o estilo externo para ficar organizado. O estilo externo também serve quando você tem vários sites e o estilo são bem parecidos.

vamos entender como funciona, no lugar de colocarmos uma tag style na cabeça do arquivo, nos vamos criar um link de css. Você só precisa ir no mesmo lugar onde criamos a tag style escrever link e vão aperecer várias opções, escolha o link css. vai auto preencher da seguinte forma: 

<link rel="stylesheet" href="style.css">

Você pode tentar entrar, mas ainda não vai funcionar. Para funcionar você terá que criar esse link,. Precisa apenas segurar o botão crtl e clicar em cima do link. Nessa hora vai aparecer uma notificação que o link não existe, agora basta clicar em criar link.

Agora tudo que você estava fazendo na cabeça do seu arquivo poderá ser feito em outro arquivo separado.

Medidas:

Medidas absolutas e relativas:

cm (centimetros), mm (milimetros), in (polegada), px (pixel), pt (ponto), pc (paica). São as medidas absolutas.

E em (Medida relativa ao tamanho M da fonte), ex (Medida relativa ao tamanho x da fonte), rem (Medida Também ralativa ao M, mas considarando o que está no CSS), vw (Medida relativa a largura da tela do dispositivo usasdo), vh (Medida relativa a altura da tela do dispositivo usado), % (Medida relativa a porcentagem). São as medidas relativas.

dica:

É recomendado usar as medidas em e px.
