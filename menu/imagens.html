Imagens no HTML:

Adicionar imagens:

tag IMG:

Para colocar imagens você precisa baixar uma imagem ou escolher alguma imagem do seu computador, e colocar junto da pasta que está seu arquivo HTML (Com um nome fácil de preferência).

Feito isso é só usar a tag <img> Essa tag é assim:

 <img src="" alt="">

Você vai colocar o nome da imagem com '.png' entre os primerios parênteses ("") (png é um dos melhores formatos que existem) e entre o segundo parêntesis ("") vai usar um parâmetro.

Uma forma fácil de selecionar qualquer imagem, link ou arquivo que você adicionou na sua pasta, junto com seu arquivo HTML é deixar o cursór no meio das aspas ("") src, segurar o 'ctrl' no teclado e precionar a teclá 'espaço'. Assim vao aparecer a lista de todos os arquivos da pasta.

Na imagm acima foi feito assim:

<img src="imagens/boneco.png" alt="Exemplo">

Tamanho de imagens:

Existem vários tipos de imagens, por exemplo o .jpg o diferencial desse formato é que tem uma auta compactação. O .png tem uma compactação um pouco menor mas permite transparência. Também tem o farmato .gif com baixa compactação, mas permite animacão e transparência.

Tamanho de imagens é um assunto muito importante, o grande professór Guanabara falou: Uma imagem de largura de 1500 megas é mais que o suficiente para usar em sites.

Obs: Quando editar o tamanho de imagens você precisa redimencionar proporcionalmente o tamanho e largura. Normalmente imagens usadas em sites tem 600 megas de tamanho, que é um tamanho ótimo.

Então é bom sempre deixar seus arquivos de imagens sempre com tamanhos pequenos, para não ser um site tão pesado.

Carregando imagens:

Podemos carregar imagens da própria pasta que fica seu arquivo HTML,como no exemplo que você viu, mas Também tem outras formas. Por exemplo, podemos fazer uma pasta de imagens dentro da própria pasta que estar o arquivo HTML. Fazendo isso não vai funcionar só colocar o nome da imagem, vai ser mais fácil usar a forma de pesquisar os arquivos segurando 'ctrl' e precionando a teclá 'espaço'.

Também dá para usar um 'link', você busca em seu navegador a imagemque quer, então clica com o segundo botão do mouse e clica em 'copiar endereço da imagem'. Usa esse link copiado no lugar onde colocaria o nome da imagem na tag <img> e essa imagem vai ser mostrada buscando direto da internet. Essa imagem vem de outro servidor, então se o servidor cair a imagem não vai ser mostrada. Será mostrado apenas o parâmetro posto na tag.

Compatível com todos os dispositivos:

Hoje se usam muitos dispositivos para navegar na internet. Celuláres, teblets e computadores são exemplos. A diferença deles são o tamanho de cada um, e para ter uma melhor foma de navegar em um site é importante ter imagens que se adaptem a cada tamanho. Como exemplo fiz três imagens de tamanhos diferentes, uma de 300 x 300, outra de 700 x 700 e a última 850 x 850. Vamos usar a tag <picture> e dentro vamos usar as tags de imagem <img>. Usamos a tag picture para informar que vai ter imagem nesse lugar do arquivo. E essa tag Também permite usar várias fontes de imagens, precisamos dessa função pois precisaremos que a imagem mude de acordo com o tamanho. Nesse exemplo, como falado, vamos usar três tamanhos. A imagem abaixo é adaptável. Pode aumentar e diminuir seu navegador para testar.

Olhando para o código acima podemos ver que foi usado a tag <picture> e em seguida a tag <source>. Mas primeiro coloquei a tag <img> com a maior imagem criada 1000 x 1000. Em seguida o visual studio code quando digitamos a iniciais da palavra source vão aparecer algumas opções, você irá escolher a source media type, quando selecionar essa opção, vai auto preencher da seguinte forma:

<source media="(min-width: )" srcset="" type="image/">

Podemos observar que auto preenche com o min-width. Foi auterado para o máximo que é a maneira que acho mais fácil de usar: max-width, logo em seguida preencha o tamamho máximo permitido para que não tenha barra de rolagem na sua imagem (Nesse caso foi de 1050px). No próximo parâmetro você colocará sua imagem média. Infelizmente não consegui buscar com o ctrl espaço, mas já foi mostrado como faz para colocar imagens. E por ultimo, no parâmetro type você vai colocar o tipo de imagem que foi usado.(No caso foi usado o png)

Observe que o source para troca de imagem de grande para média foi em cima da tag <img>. O source da imagem pequena será em cima do source anterior, como demonstrado no exemplo. Será da mesma forma, mas com o whith máximo 700px e a imagem que sera posta será a pequena que foi renomeada assim: p.png. E está na pasta imagens.

Fivecon:

Como mudar o fivecon do seu site:

O formato da imagem tem que ser '.ico' , você pode achar várias imagens nesse formatos na web. O iconarchive.com é um site que tem uma quantidade enorme de ícones para escolher. Se você for criativo e paciente, dá pra você mesmo fazer seu ícone do seu site. No site favicon.cc dá pra fazer seu ícone, é um pouco trabalhoso, mas sai da forma que desejar. Tem mais uma forma de fazer seu ícone, no site fivecon.io. Nesse site dá pra você fazer um texto, uma imagem ou até mesmo um emoji você transformar em um ícone.

Para adicionar seu ícone basta ir na cabeça do HTML (Na tag <head>), em cima da tag <tittle> escrava 'link' nessa hora vai aparecer uma lista com vários tipos de links que podem ser usados. Escolha a opção 'favicon', vai preencher automaticamente o comando, agora é só apagar dessa linha autopreenchida o que está dentro das aspas 'fivecon.ico', então é só apartar 'ctrl''espaço' no seu teclado e usar o ícone que quiser.
