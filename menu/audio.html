Usar áudio:

Como adicionar:

Podemos adicionar áudios no nosso site também, usando a tag <audio>. Aqui vai o exemplo:

<audio src="exemplodeáudio.mp3">

Assim seu áudio será inserido no seu arquivo, mas não vai tocar. Para tocar você pode colocar os parâmetros autoplay que vai iniciar o áudio assim que o site for carregado ou o controls que mostrará um controle de áudio.

Quando for usado o autoplay temos outra configuração que pode ser interesante na hora de colocar uma música de fundo no seu site. o parâmetro loop faz o seu áudio rainiciar sem parar, é bem comum ser usado quando o áudio é curto. Ex:

<audio src="áudios/áudio1.mp3" autoplay loop>

Dependendo do navegador o áudio não irá funcionar, mas temos como resolver esse problema usando autros formatos de áudio. Temos os formatos OGG, WAV e o que usamos de exemplo MP3. Você converte o áudio para esses três tipos e coloca no seu arquivo. Assim, se o navegardor não for conpativel com um, será com o outro. Para colocar os três formatos temos que usar o type. Quando usamos a tag <audio> ela se auto preenche com o src.

<audio src=""></audio>

Apagamos esse src e vamos dar um enter entre as tags de abertura e fechamento para colocarmos o source. Vamos colocar manualmente o source. Vimos na parte de imagens no HTML que enquanto estamos digitando a palavra source no seu visual studio code, vão aparecendo várias opções e você irá escolher a opção source src.

Agora preencha com os áudios que você converteu e o tipo de cada um.

Para saber qual configuração usar em cada media type basta buscar nesse link o tipo de midia que irá usar. Clica no link, então no teclado aperta ctrl f vai aparecer uma barra de busca e é só buscar o tipo de arquivo que está usando clicando no enter. Obs: Para navegar no site em busca do que você quer, depois que pesquisou o tipo do arquivo, basta ficar clicando no enter até achar.

Agora que está adicionado os princpais tipos de áudios colocaremos as pré configurações na tag áudio já citadas acima. E se mesmo assim o áudio não for iniciado colocaremos uma forma dele ser baixado. Dessa forma o áudio será apresentado para quem está navegando no site de alguma maneira.

Mais uma dica sobre tamanho de arquivo: Evite usar o tipo WAV de áudio porque mesmo sendo Compatível ele é muito grande e pode deixar seu site bem pesado.
